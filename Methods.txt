Method: 
------
A method is a block of code which performs a specific task. 
Methods are created inside a class. 

Method creation Syntax:
-----------------------
accessmodifier returntype methodname(arguments){

  	// method body
  }

accessmodifier: 
The access modifer tells about the access level of the method.
Java supports 4 different access modifiers:

1. public 2, private 3.protected 4. default

public - available through oout the application
private - available only in the specified class. 
protected - 
default - 


return type: the return type of the method tells about the type of the value which is returned by the method. 
If the method does not return any anything we use a key


methodName: methodName is the name of the method
